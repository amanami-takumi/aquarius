### 動作確認メモ
1. `app` と `backend` ディレクトリで `npm install` を実行。
2. `docker-compose up` で Postgres/Minio を同時起動。
3. フロントエンド: `npm run dev` で起動し、タブ切り替え・Markdown保存を確認。
4. バックエンド: `npm run dev` または `npm run start` で起動。`curl http://localhost:5000/healthz` が `{"status":"ok"}` を返すことを確認。
5. エディタで入力して1秒程度待ち、バックエンドログに `/documents` へのPOSTが出力されることを確認。
